# 🚖 Uber Data Analysis Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard created to analyze Uber trip data and identify booking trends, revenue insights, and customer behavior.

---

## 🛠 Tools Used

- Power BI
- Excel
- DAX

---

## 📊 Dashboard Preview

![Dashboard](dashboard.png)

---

## 📈 Key KPIs

- Total Bookings
- Total Revenue
- Average Trip Distance
- Average Trip Time
- Booking Status
- Payment Method
- Vehicle Type Analysis

---

## 📍 Key Insights

- Peak booking hours
- Most preferred vehicle type
- Revenue by payment method
- Trip distance distribution
- Booking trend analysis
